## Beschreibung

> Liefer die Anzahl von Vokalen in einer definierten Zeichenkette (`String`).
> 
> Vokale sind `a`, `e`, `i`, `o`, `u`.
> 
> Der Eingabetext (`String`) wird nur aus Kleinbuchstaben und/oder Leerzeichen bestehen.
